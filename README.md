# Monster-Fighting-UI
This is a simple game of user vs monster fight. There are four buttons the user can press, attack, strong attack, heal and log. Before the game starts, it will let the user choose the max life for the monster and the user. It has implemented a checker to make sure the user has entered a valid number.
